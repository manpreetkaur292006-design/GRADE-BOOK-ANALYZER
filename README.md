# Overview

Grade-Book-Analyzer is a Python program that helps manage and analyze student grades.
It allows data entry manually or from a CSV file, then calculates average, median, highest and lowest marks, assigns grades, and displays results in a clear table.

---

# Features

- Input data manually or from a CSV file
  
- Calculates average and median
  
- Finds students with highest and lowest scores
   
- Assigns letter grades (Aâ€“F)
  
- Shows grade distribution and pass/fail summary
  
- Option to analyze multiple datasets  

---

# Usage

1. Run the program:
   
python Grade-Book-Analyzer.py

2. Choose data input method (1 = manual, 2 = CSV).  
3. View summary, grade distribution, pass/fail list, and result table.
